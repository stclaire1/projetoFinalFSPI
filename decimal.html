Sistema Decimal

O sistema decimal é o mais utilizado no dia a dia. Nele utilizamos dez dígitos diferentes (0 a 9) para representar valores. A posição de cada dígito em um número decimal indica o seu valor relativo. Por exemplo, o número 257 tem o 2 na posição das centenas, o 5 na posição das dezenas e o 7 na posição das unidades.

A conversão de números decimais para as bases binárias e hexadecimais segue a mesma lógica utilizada na conversão de números de outros sistemas numéricos para o sistema decimal. Por exemplo, ao converter números binários para decimais, é necessário multiplicar cada dígito binário pela potência de 2 correspondente à sua posição e somar os resultados. De forma similar, na conversão de um número hexadecimal para decimal, multiplica-se cada dígito hexadecimal pelo valor correspondente e soma-se os resultados obtidos.

Conversões entre Sistemas

Decimal para Binário

Divida o número decimal por 2 sucessivamente até chegar a 0, registrando o resto de cada divisão. O número binário é formado pelos restos, lidos em ordem reversa, do último para o primeiro.

Exemplo: Converter o número decimal 45 para binário.

Passo 1: Dividir o número por 2 sucessivamente até chegar a 0.

45 ÷ 2 = 22, resto 0

22 ÷ 2 = 11, resto 1

11 ÷ 2 = 5, resto 0

5 ÷ 2 = 2, resto 1

2 ÷ 2 = 1, resto 0

1 ÷ 2 = 0, resto 1

Passo 2: Registrar os restos em ordem reversa. Os restos são 101010, lidos de trás para frente.

Portanto, o número decimal 45 é igual a 101101 em binário.

Decimal para Hexadecimal

Para a conversão de decimal para hexadecimal, dividimos o número decimal por 16 sucessivamente até chegar a 0, registrando o resto de cada divisão. Os restos de 10 a 15 são representados pelas letras A, B, C, D, E e F, respectivamente. Os demais restos são representados pelos próprios números.

Exemplo: Converter o número decimal 438 para hexadecimal.

Passo 1: Dividir o número por 16 sucessivamente até chegar a 0.

Passo 2: Registrar os restos em ordem reversa. Note que o resto da segunda divisão foi o número 11, que corresponde ao número B em Hexadecimal. .Portanto, o número decimal 438 é igual a 1B6 em hexadecimal.
